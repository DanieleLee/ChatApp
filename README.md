# ChatApp# Node.js Express을 활용한 채팅 프로그램
## 사용Skills: 
####         1. Node.js , Express
####         2. View : BootStrap, Semantic 
####         3. 데이터베이스 :MongoDB


## 구현기능: 
####         1. 그룹채팅- 채팅방을 만들어서 채팅방에 입장한 인원끼리 채팅을할수있다. 
####         2.          채팅방내 인원끼리 일대일채팅을 할수있다.
####         3. 개인채팅- 소켓에 연결된 사람끼리 채팅을 할수있다.
